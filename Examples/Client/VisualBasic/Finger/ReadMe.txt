GENERAL INFORMATION

	The Ultimate TCP/IP Finger Control 3.0 is a robust component 
that allows developers to quickly add Finger capabilities to their 
applications.  

********************************************************************

FEATURES
	
	- ability to set connection and receive timeouts. 
	- rich set of enumerated error constants. 
	- optional storage of returned finger data to a text file
	- ability to toggle between synchronous and asynchronous modes.
	- cancellation of an asynchronous finger operation.   

********************************************************************

COMPATIBILITY	

	This demo was written using Visual Basic 6. No VB 6 specific
functions were used.

********************************************************************

HOW TO USE THE DEMO

	1) Enter a valid Finger domain name.  The syntax 
           is: [fingertextfile]@domain.com.
	
	2) Click on the Finger button.
	
	3) The status textbox indicates the status of the operation by using 
	   the returned finger errorcodes (FingerErrorCodes enumeration). 
	
	4) The results of the finger operation are displayed in the bottom
           textbox.

Note: [] indicates optional syntax element.
