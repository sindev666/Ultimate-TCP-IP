***********************************************************************

General Features
	
	- the Imap4 3.0 Control exposes over 100 properties, methods
	  and events. 

	- create, rename and delete mailboxes with one procedure call.

	- ability to toggle asynchronous and synchronous operations on
	  and off with the BlockingMode property.

	- 27 events (OnMessageFetch(), OnMailBoxSelect(), etc.)

***********************************************************************

Compatibility Issues

	- this was written with VB 6.0.  It utilizes standard controls, 
	  the Dundas History and Imap4 3.0 controls as well as the 
	  listview and treeview controls from the Microsoft Common 
	  Controls 5.0 (SP2) ocx. One VB 6 specific function called is 
	  the Split() function.

***********************************************************************

How to Use This Demo

	1) enter a valid username, userpassword and hostname of the Imap4 
	   server and connect to server.
	
	2) use the menu to manipulate mailboxes.

	3) use the tooltiptext.
